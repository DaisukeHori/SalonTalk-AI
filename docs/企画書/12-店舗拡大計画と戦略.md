## 12. 店舗拡大計画と戦略

### 12.1 フェーズ別拡大計画

本プロジェクトは**自社店舗（Prestoグループ）でのテスト・検証**を経て、**外部美容室への販売展開**を行う。

#### フェーズ0：自社テスト（2026年1月〜3月）

| 項目 | 内容 |
|------|------|
| 対象 | 美容室Prestoグループ（5店舗） |
| 目的 | 製品検証、UX改善、成功事例の創出 |
| KPI | 店販売上+20%、スタッフ満足度80%以上 |
| 売上 | なし（自社利用） |
| 投資 | 開発費45万円 + 運用費25万円 = 70万円 |
| 開発期間 | **3ヶ月**（AI駆動型開発） |

**フェーズ0の成果物:**
- 定量的な効果実績（店販売上向上率、離職率変化）
- 導入事例・スタッフインタビュー動画
- 製品の完成度向上

#### フェーズ1：初期外部展開（2026年4月〜12月）

| 項目 | 内容 |
|------|------|
| 目標店舗数 | 17店舗 |
| ターゲット | Revol取引先美容室、DX意識の高い中規模サロン |
| 獲得方法 | パートナー経由、直販営業、紹介 |
| 月間獲得目標 | 2店舗/月（9ヶ月で17店舗） |
| 年間売上 | 約770万円 |

#### フェーズ2：成長期（2027年）

| 項目 | 内容 |
|------|------|
| 目標店舗数 | 50店舗（+33店舗） |
| ターゲット | 地方展開、チェーン店グループ |
| 獲得方法 | パートナー経由60%、直販40% |
| 月間獲得目標 | 3店舗/月 |
| 年間売上 | 約2,300万円 |

#### フェーズ3：拡大期（2028年〜2030年）

| 年度 | 目標店舗数 | 月間獲得 | 年間売上 |
|------|-----------|---------|---------|
| 2028年（3年目） | 100店舗 | 4店舗/月 | 4,300万円 |
| 2029年（4年目） | 180店舗 | 7店舗/月 | 7,700万円 |
| 2030年（5年目） | 300店舗 | 10店舗/月 | 1億2,900万円 |

### 12.2 拡大戦略（5つの柱）

#### 戦略1：自社成功事例の徹底活用

**施策:**
- Prestoグループでの成功事例を詳細にドキュメント化
- ビフォー・アフターの定量データ（店販売上、離職率、育成期間）
- スタッフ・経営者インタビュー動画の制作
- 「自社で使って効果を実証済み」という信頼性

**期待効果:** 信頼性向上、商談時の説得力強化

#### 戦略2：美容業界パートナー連携

**ターゲットパートナー:**

| パートナー種別 | 具体例 | 連携内容 | 手数料目安 |
|---------------|--------|----------|-----------|
| 美容ディーラー | 地域の美容商材卸 | 既存顧客への紹介 | 売上の15〜20% |
| POSベンダー | サロンアンサー、Bionly等 | システム連携・共同販売 | 売上の10〜15% |
| 美容メディア | 女性モード社、美容経済新聞 | 記事掲載・セミナー共催 | 広告費 |
| 美容コンサル | 経営コンサルタント | クライアント紹介 | 売上の20% |

**参考事例:** 
- サロンアンサー（エクシードシステム）: 導入実績8,000店舗以上（[出典](https://ex.salonanswer.com/column/salon-management-success/)）
- 美容経済新聞社×女性モード社: バーティカルSaaS強化のための資本業務提携（[出典](https://prtimes.jp/main/html/rd/p/000000030.000046762.html)）

#### 戦略3：コンテンツマーケティング

**施策:**

| チャネル | 内容 | 頻度 | 担当 |
|---------|------|------|------|
| YouTube | 店販トークスキル解説、導入事例 | 週1本 | AI生成+編集 |
| Instagram | 成功事例ショート、使い方Tips | 毎日 | AI生成 |
| note/ブログ | 美容師向けノウハウ記事 | 週2本 | Claude執筆 |
| メルマガ | 業界動向、新機能紹介 | 週1回 | 自動化 |

**SEO/MEO対策:**
- 「美容室 店販 売上アップ」「美容師 育成 AI」等のキーワードで上位表示
- Googleビジネスプロフィール最適化

**参考:** 美容室選びの約6割がWeb検索・予約サイト経由（[出典](https://cs.valuedesign.jp/column/beautysalon_attractcustomers)）

#### 戦略4：リファラル（紹介）プログラム

**プログラム設計:**

| 紹介者 | 報酬 | 条件 |
|--------|------|------|
| 導入済み店舗オーナー | 1ヶ月無料 or 3万円キャッシュバック | 紹介先が契約締結 |
| 美容師個人 | Amazonギフト1万円 | 紹介先が契約締結 |
| パートナー企業 | 月額の15〜20% | 継続報酬 |

**根拠:** 
- 92%の消費者が友人・家族の口コミを信頼（[出典](https://stylespeak.com/marketing-strategies-to-help-salons-acquire-new-clients-and-retain-existing-ones/)）
- 美容業界は口コミ効果が特に高い

### 12.3 店舗獲得のKPIと進捗管理

#### フェーズ1（17店舗達成）の詳細KPI

| 指標 | 目標値 | 計測方法 |
|------|--------|---------|
| 月間リード獲得数 | 20件 | CRM管理 |
| 広告費 | 50万円/月 | Google/Meta広告 |
| 商談化率 | 25% | 商談数/リード数 |
| 成約率 | 40% | 成約数/商談数 |
| 月間成約数 | 2店舗 | 実績カウント |
| 解約率（チャーン） | 3%以下 | 月次計測 |
| NPS（推奨度） | 40以上 | 四半期調査 |

#### 顧客獲得コスト（CAC）目標

| フェーズ | CAC目標 | 内訳 |
|---------|---------|------|
| フェーズ1 | 25万円以下 | 広告費50万円÷2件 |
| フェーズ2 | 20万円以下 | パートナー手数料含む |
| フェーズ3 | 15万円以下 | 紹介比率向上 |

**LTV（顧客生涯価値）:**
- 月額3.8万円 × 24ヶ月（平均継続期間）= **91.2万円**
- LTV/CAC比率 = 91.2万円 / 25万円 = **3.6倍**（目標: 3倍以上）

### 12.4 チャネル別 店舗獲得計画

| チャネル | フェーズ1（17店舗） | フェーズ2（+33店舗） | フェーズ3（+50店舗/年） |
|---------|-------------------|---------------------|----------------------|
| 直販（自社営業） | 5店舗（29%） | 8店舗（24%） | 10店舗（20%） |
| 既存顧客紹介 | 2店舗（12%） | 5店舗（15%） | 10店舗（20%） |
| パートナー経由 | 10店舗（59%） | 20店舗（61%） | 30店舗（60%） |
| **合計** | **17店舗** | **33店舗** | **50店舗** |

**補足:**
- フェーズ1はパートナー経由が主力（美容ディーラー、POSベンダー連携）
- 直販は「確実に取れる」取引先・知人関係に限定
- 紹介は自社Prestoグループでの成功実績を見た口コミに期待

### 12.5 組織体制（フェーズ別）

| フェーズ | 体制 | 役割 |
|---------|------|------|
| フェーズ0-1 | PM 1名 | 開発・営業・CS全て |
| フェーズ2 | +営業1名、CS1名 | 営業専任、カスタマーサクセス専任 |
| フェーズ3 | +パートナー担当1名 | 代理店開拓・管理 |

**注:** 人員追加は売上・利益に応じて段階的に実施。AI活用により最小限の人員で運営。
