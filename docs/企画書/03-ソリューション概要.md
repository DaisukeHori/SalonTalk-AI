## 3. ソリューション概要

### 3.1 プロダクトビジョン

**「売れるトーク」を科学し、全スタッフの武器にする**

セット面にiPadを設置するだけで：
1. 会話を**完全オンデバイスで文字起こし**（プライバシー保護）
2. **美容師とお客様の発話を自動分離**
3. **7つの科学的指標でトーク品質をスコアリング**
4. **リアルタイムで最適な提案タイミングを通知**
5. **トップスタイリストの成功パターンをレコメンド**
