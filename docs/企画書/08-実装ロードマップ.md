### 8.2 第1フェーズ：初期版開発（2026年1月〜3月）

**目標**: 基本機能を実装し、3店舗でβテスト

| 週 | 開発内容 |
|----|---------|
| 1-2週 | React Nativeプロジェクト構築、Supabase設定 |
| 3-4週 | Apple SpeechAnalyzer統合、リアルタイム文字起こし |
| 5-6週 | AssemblyAI統合、話者分離パイプライン |
| 7-8週 | Claude Sonnet 4.5統合、基本分析 |
| 9-10週 | iPad UI実装、リアルタイムスコア表示 |
| 11-12週 | βテスト開始、フィードバック収集 |

**成果物**:
- 動作する初期版
- βテスト店舗3店舗確保（Revol取引先から選定）
- 初期フィードバック収集

**予算**: 2,000万円

### 8.3 第2フェーズ：AI分析強化（2026年4月〜9月）

**目標**: AI分析精度を向上し、10店舗に拡大

| 月 | 開発内容 |
|----|---------|
| 4月 | Qdrant構築、成功事例DBスキーマ設計 |
| 5月 | 類似事例検索機能、ベクトル埋め込み |
| 6月 | リアルタイムAIアシスト（提案タイミング通知） |
| 7月 | スコアリング精度チューニング |
| 8月 | レポート自動生成の高度化 |
| 9月 | 10店舗への展開、データ蓄積 |

**成果物**:
- リアルタイムコーチング機能
- 成功事例DB（初期データ100件）
- 導入店舗10店舗

**予算**: 3,000万円
