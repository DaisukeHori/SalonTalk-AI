### 7.3 投資回収シミュレーション

#### モデルケース：中規模サロン（スタッフ10名）

**導入前の状況**：
- 月間施術客数: 300名
- 店販購入率: 15%（45名購入）
- 平均単価: ¥3,500
- 月間店販売上: ¥157,500

**導入後の効果（保守的シナリオ：+20%）**：

| 項目 | 導入前 | 導入後 | 差額 |
|------|--------|--------|------|
| 店販購入率 | 15% | 18% | +3pt |
| 月間購入者数 | 45名 | 54名 | +9名 |
| 月間店販売上 | ¥157,500 | ¥189,000 | **+¥31,500** |

**追加効果：育成期間短縮**
- 育成期間: 12ヶ月 → 6ヶ月（50%短縮）
- 育成コスト削減: 7.5万円/人
- 年間効果（新人2名想定）: 15万円 → 月換算1.25万円

**投資対効果（ROI）計算**：
```
月間効果 = 店販増収（3.15万円）+ 育成コスト削減（1.25万円）= 4.4万円
月間コスト = 2.98万円
月間純利益 = 1.42万円

初期費用回収期間 = 15万円 / 1.42万円 = 約10.6ヶ月
年間投資対効果 = (1.42万円 × 12) / (15万円 + 2.98万円 × 12) × 100 = 32.5%
```

**現実的シナリオ（+35%、Siro平均値）の場合**：
```
月間店販売上: 15.75万円 → 21.26万円（+5.51万円）
月間効果 = 5.51万円 + 1.25万円 = 6.76万円
月間純利益 = 6.76万円 - 2.98万円 = 3.78万円
初期費用回収期間 = 15万円 / 3.78万円 = 約4ヶ月
年間投資対効果 = 126%
```
